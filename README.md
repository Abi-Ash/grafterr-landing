# Grafterr Landing Page

This project is a responsive landing page built as part of a frontend technical assessment. The goal was to replicate the provided design and implement dynamic content loading using Vanilla JavaScript.

## Tech Stack

* HTML5
* CSS3 (Flexbox, Grid, CSS variables)
* Vanilla JavaScript (ES6)

## Features

* Pixel-based layout based on the provided Figma design
* Dynamic content loaded from a local JSON file
* Simulated API delay using `setTimeout`
* Loading skeleton displayed while data loads
* Responsive layout (mobile, tablet, desktop)
* Product carousel with previous and next navigation
* Error handling with retry option if data fetch fails

## Project Structure

grafterr-landing

* index.html
* css/
* js/
* data/content.json
* assets/images

## Setup Instructions

1. Clone the repository
2. Open `index.html` in a browser
